# Mr.Snake

Cílem hry je přežít co nejdéle v nehostinné poušti. Každý den je vizuálně znázorněn změnou barvy pozadí – čím tmavší je pozadí, tím hlubší je noc. Během noci se nepřátelé, reprezentovaní modrými a fialovými čtverečky, pohybují rychleji, čímž se hra stává náročnější.

Stejně jako v klasické hře Snake, sbíráte červené čtverečky, které způsobují, že váš had roste. Musíte se vyhýbat překážkám a nepřátelům, plánovat každý pohyb a přežít co nejvíce dní v tomto drsném prostředí.
